PALACE DRUM CLINIC

Drum Clinic Application

Designing & building a website that shows live and exclusive drumming content to subscribers

Overview

PDC Application is an app for percussion fans. The app aims to host exclusive content made by popular drummers, with the intention to help teach beginners through to experts in order to improve their drumming skills.

My role: I was the sole designer from user research through to the user interface
Timeline: September 2022 - February 2023
Tools: Google Surveys, Figma, Miro

The Challenge

To design a website and application for subscribers to a drum clinic. The website will include exclusive video content & access to unique features: live video, profile & exclusive drumming events.

Problem

The Palace Drum Clinic is a charity that aims to provide educational workshops and lessons to schools and organisations. Plenty of world renown drummers from famous bands work with the drum clinic, hence their slogan ‘Be part of Drumming Royalty’ – these drummers create exclusive content for the clinic and attend both virtual and in-person events.
The current website does not have features to allow for subscribers to access exclusive content. We want to create a website and mobile app that paying subscribers can use to view content, access live lessons, contact drummers and see upcoming events.

Hypothesis

An easy to use website and app will increase reach and the audience for the Palace Drum Clinic, improve their drumming ability and increase attendees of drumming events.

Solution

Design and build a website that will extend the current PDC website. This website will be built using React to allow for more exclusive and adaptable features to be created (rather than the standard Wordpress website). Users will be able to create a personalised account, view and save pre-recorded and live video and purchase tickets for events.

Research & Discovery

Competitor Analysis

Together with members of the PDC, we decided who the main competitors are. Youtube offers thousands of free drumming videos. Drumeo is a paid service that specialises in online drumming lessons. I analysed the positives and negatives of both competitors.

User Research

I wanted to better understand who the users I was designing for would be.
1. Understand users knowledge and opinions of current drumming applications.
2. What features would they find useful.
3. What friction points do drumming learners currently have.
I surveyed those that currently work for the drum clinic and a selection of those that are beginners to drumming or are intermediate learners (54 users). I gave these users 10 questions to gain quantitative insights.

95% Use Youtube as their prime educational resource for drumming related topics

40% Take drumming lessons with a tutor virtually

66% Would like to attend more drumming related events

Only 20% could name a competitor that was not YouTube

User Persona

A primary user persona was created using the user research information.

User Stories

As a user, I want to be able to study drumming and consume content anywhere, so I must be able to download videos so they are available offline and on demand.
As a paying user, I want to have access to the most up to date and curated drumming content that is available.
As a user who is new to drumming, I want to be able to access beginner's drumming content and feel my ability has progressed after watching it.

As a user, I want to be able to access content from desktop and mobile.

Style Guide

Whilst working at the BBC, I have learnt the importance of having a style / design system. The BBC have the 'Global Experience Language', a system of reusable interaction patterns. Even though I was working solo on this project, I knew that creating a mini design system of my own will assist with saving time, keeping things consistent and improve the overall look of the project and be very beneficial for the developer building the application! As a developer myself, I am biased to the last point in particular - having repeatable components makes developing an application much less painful and time consuming.
